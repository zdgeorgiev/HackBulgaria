# K Table

Implement a function in [Java](KTable.java), [Python](k_table.py)
or [C++](k_table.cpp) that given a sorted table (2-dimensional array)
and an integer `k` it finds the k-th smallest element in the table.

## Example

Input:
```
 1  5 10 13
14 17 22 35
15 23 31 47
25 27 36 54
```

Outputs:
```
1-st smallest: 1
3-rd smallest: 10
6-th smallest: 15
9-th smallest: 23
16-th smallest: 54
```
